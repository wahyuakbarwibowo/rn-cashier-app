Panduan Instalasi & Penggunaan: Aminmart Cashier

Terima kasih telah memilih Aminmart Cashier sebagai mitra bisnis Anda. Berikut adalah panduan langkah demi langkah untuk menginstal dan mulai menggunakan aplikasi di perangkat Android Anda.

Karena aplikasi ini didistribusikan melalui itch.io, Anda perlu mengikuti langkah-langkah berikut untuk menginstal file APK secara manual:

  1. Download APK
    Unduh file aminmart-cashier.apk langsung dari halaman itch.io kami.
  2. Aktifkan "Unknown Sources" (Sumber Tidak Dikenal)
    Buka menu Settings > Security atau Privacy di ponsel Anda, lalu aktifkan opsi "Install apps from Unknown Sources".

    *Catatan: Pada versi Android terbaru, Anda mungkin diminta memberikan izin khusus untuk browser atau file manager Anda.

  3. Instal Aplikasi
    Cari file APK yang sudah diunduh menggunakan File Manager, klik pada file tersebut, dan pilih Install.
  4. Buka Aplikasi
    Setelah selesai, ikon Aminmart Cashier akan muncul di daftar aplikasi Anda. Klik untuk memulai!

Agar aplikasi dapat berfungsi maksimal dengan printer thermal Anda, lakukan langkah berikut:

1. Hubungkan Printer Bluetooth

  • Nyalakan Printer Bluetooth Thermal Anda (58mm).
  • Buka menu Bluetooth di pengaturan HP dan lakukan Pairing dengan printer Anda.
  • Buka aplikasi Aminmart Cashier, masuk ke menu Pengaturan Printer, dan pilih nama printer yang sudah terhubung.

2. Input Data Produk

  • Masuk ke menu Inventaris untuk menambahkan produk retail (snack, barang fisik, dll).
  • Gunakan fitur Scan Barcode untuk mempercepat input data stok.

3. Konfigurasi Layanan Digital (PPOB)

  • Buka menu Custom Category untuk menentukan layanan digital apa saja yang ingin Anda jual (Pulsa, Token PLN, dll).
  • Input harga modal dan harga jual untuk otomatisasi perhitungan laba.
